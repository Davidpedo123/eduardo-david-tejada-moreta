Kit de Arquitectura Backend

Este repositorio consiste en una maqueta para el inicio de cualquier proyecto, la diferencia de este es que está enfocado en que el dev se concentre totalmente en la lógica del servicio, obviando la creación de APIs Gateway desde cero o el despliegue del proyecto y orientado a una arquitectura de microservicios.

Maqueta Microservicios

Este repositorio demuestra una arquitectura de microservicios donde los servicios están diseñados utilizando FastAPI para el backend y Angular para el frontend. La comunicación y el enrutamiento de las solicitudes se gestionan a través de un proxy inverso Nginx, mientras que la orquestación y el despliegue de los contenedores se manejan con Docker Compose.
